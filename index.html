快速上手

导入 jquery1.9.js 和 viewimg.1.0.js

 <script type="text/javascript" src="js/jquery-1.9.1.min.js" ></script>
 <script type="text/javascript" src="js/viewimg.1.0.js" ></script>

创建一个文件上传控件 一个显示图片的容器

<input type="file" id="myfile" />
<div id="localImag"></div>

设置容器css样式

#localImag {
	width: 200px;
	height: 200px;
	background : #5F748A;
	*overflow: hidden;  /** IE6 必须 **/
}

设置显示图片的参数

$(function() {
		var file = $("#myfile");
		file.viewimg({
			targetDiv : $("#localImag"),		//图片显示的容器
			sizingMethod : "scale",			//按照图片比例自适应容器
			errorDiv : $("#error")			//错误显示位置
		});
	});

※sizingMethod : scale|img
scale : 按照图片比例适应外层容器的宽高
img : 显示图片原尺寸

常用参数

所有viewimg可配参数

参数名称 默认值 说明 示例
targetDiv 无 显示图片的容器(必选) targetDiv : $("#my_img")
error alert显示 错误显示方式
1.传递$("xxx")对象,错误显示在$("xxx")中
2.传递function对象,则为错误回调函数
3.不传,使用alert弹出		 error : $("#error")
error : function(msg){}
error : ""
allowTypes [ ".jpg", ".png", ".bmp", ".gif", ".jpeg" ] 允许上传的图片格式 allowTypes : [ ".png" ]
sizingMethod scale 图片显示方式
img : 全尺寸显示
scale : 适应外层容器
 sizingMethod : "scale"
ie_delay 2000 针对IE 789 的问题 图片过大的 需要延迟执行 ie_delay : 2000
